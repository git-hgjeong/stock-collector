# 검색식 종목 수집기

특정 검색식에 대한 종목 정보를 수집하고 API Server로 수집된 데이터를 전송한다.

## Getting Started

```
python collector.py
```

### Prerequisites
* 키움증권(http://www.kiwoom.com)의 Open API+, KOA Studio를 설치.
* 파이썬 3.8버전 설치.

### Installing

* collector.py 소스 안에 조건 검색식 이름과 API Server의 주소를 자신의 것으로 변경.

* 파이썬 라이브러리 설치
```
pip install pyqt5
```

```
pip install pandas
```

```
pip install requests
```

```
pip install openpyxl
```
